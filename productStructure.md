Generated or local-only directories such as `.git/`, `.pytest_cache/`, `__pycache__/`,
and built frontend assets are intentionally omitted unless they describe a runtime surface.

Runtime state is stored outside the repo under `~/.nullifier/` by default:

- `config.toml` for user routing/cache/R/PAML overrides.
- `runs.db` for run history and event replay.
- `flags.db` for classification corrections, configurable via `flags.db_path`.
- `ensembl_cache.db` for Ensembl API responses, default 30-day TTL.
- `gene_sets_cache.pkl` for parsed SynGO/BBB/control sets, default 7-day TTL.
- `gnomad_cache.db` for gnomAD constraint responses, default 30-day TTL.
- `paml_cache.db` for codeml branch-model omega results, default 90-day TTL.
- `rdnds_cache.db` for R/seqinr pairwise dN/dS results, default 90-day TTL.
